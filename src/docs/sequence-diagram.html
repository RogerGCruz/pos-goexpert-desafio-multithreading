Legenda dos Requisitos

  • Requisito 1 — Requisições Simultâneas: duas goroutines lançadas ao mesmo tempo para BrasilAPI e ViaCEP
  • Requisito 2 — Race Condition: select retorna a primeira resposta bem-sucedida e descarta a outra
  • Requisito 3 — Output: JSON no terminal com o nome da API vencedora e o endereço completo
  • Requisito 4 — Timeout: context.WithTimeout de 1 segundo; se disparado, retorna ErrTimeout
